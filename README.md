# Installation
1. Open the extensions sidebar on Visual Studio Code
2. Search for emerald-nebula theme
3. Click Install
4. Click Reload to reload your editor
5. Select the Manage Cog (bottom left) > Color Theme > emerald-nebula Theme
6. ⭐⭐⭐⭐⭐ Rate five-stars 😃
